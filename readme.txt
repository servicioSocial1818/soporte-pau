Como instalar el sistema
1. Contar con un SGBD (Sistema gestor de base de datos) (SQL)
2.- Crear una base de datos llamada soportetec
3.- Importar la base de datos desde el archivo (BD/soportetec.sql). 
4.- Para realizar la conexion con la base de datos se debe configurar
    esta con los datos del servidor en el archivo (clases/Conexion.php)
5.- Sustituir los datos por los del servidor, usuario, contrase単a, bd.
6.- guardar los cambios 
7.- Ingresar al sistema. 
8.- Se puede ingresar mediante los usuarios preeviamente registrados segun 
    como se quiera iniciar sesion:
    ADMINISTRADOR:  
        usuario: admin 
        contrase単a: admin 
    Cliente: 
        usuario: adminCliente
        contrase単a: admin
9.- Se puede tener acceso total de la base de datos desde el servidor, por si se hace algun mal uso del sistema (Borrar usuarios, administradores, etc)
